# Local model files for semantic matching

Semantic and Hybrid matching modes load model files from this directory (browser local files served by Vite/static hosting).

Expected model path:

`/models/onnx-community/all-MiniLM-L6-v2-ONNX`

So the repository-relative directory should be:

`public/models/onnx-community/all-MiniLM-L6-v2-ONNX`

Place the full Transformers.js-compatible model assets there (config/tokenizer/onnx files).
